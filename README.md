# alice-testing-example

Пример функционального тестирования [навыка Попугай](https://github.com/yandex/alice-skills/tree/master/node.js/parrot/now) 
для Яндекс Алисы.

### Установка
```
npm i
```

### Запуск тестов
```
mocha test.js
```

### Результат
```
  ✓ should get hello on enter
  ✓ should reply the same message

  2 passing (37ms)
```
